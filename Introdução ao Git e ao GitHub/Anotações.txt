Git criado em 2005 por Linus Torvalds.
Um versionador de códigos.
Git e GitHub são coisas diferentes, apesar de aparecem e serem estudados juntos em diversos conteúdos.

Benefícios do uso de Git e GitHub:
1. Controle de versão
2. Armazenamento em nuvem
3. Trabalho em equipe
4. Melhorar o código
5. Reconhecimento

Navegação via commandline interface e instalação: comandos básicos para um bom desempenho no terminal

A forma de interação é por meio de linha de comando

Nesse curso da DIO aprendemos como mudar de pastas, listar pastas, criar pastas/arquivos e/ou deletar pastas/arquivos.

Comandos Windows:

- cd
- dir
- mkdir
- del/rmdir


Comandos Linux:

- cd
- ls
- mkdir
- rm-rf
